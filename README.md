# SkillCraft Technology

This repository contains a simple scraper project built with Python.

## Contents

- `scraper.py` - main scraping script
- `products.csv` - sample or output data file
- `requirements.txt` - Python dependencies for the project

## Setup

1. Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
python -m pip install -r requirements.txt
```

## Usage

Run the scraper:

```powershell
python scraper.py
```

## Notes

- Keep the `.venv` directory out of version control
- Update `requirements.txt` if new packages are added
